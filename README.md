# Парсер расписания электричек

Программа для парсинга расписания электричек с сайта Туту.ру со станции Москва Ярославская.

## Функциональность

- Извлекает время отправления электричек
- Определяет маршрут в формате "Откуда → Куда"
- Фильтрует рейсы по типам дней: будни или ежедневно
- Выводит результаты в консоль
